1、功能说明
    1、配置三个触摸按键为硬件检测模式，MCU依次进入SLEEP、STOP0、STOP2模式，可通过任意触摸按键唤醒

2、使用环境

    软件开发环境：KEIL MDK-ARM V5.26
    硬件环境：
        1、基于全功能板N32G457QEL-EVB开发

3、使用说明
    
    系统配置；
        1、时钟源：HSE+PLL
        2、系统时钟：16MHz
        3、TSC端口：
            PC6:CH8     PC7:CH9
        4、LED控制端口：PB10
        5、UART端口：PA9（115200，8bit data，1bit stop）

    使用方法：
        1、在KEIL下编译后烧录到全功能板，通电，D6闪烁三次后进入SLEEP模式
        2、按下任意触摸按键，唤醒MCU，D6闪烁一次后，进入STOP0模式
        3、按下任意触摸按键，唤醒MCU，D6闪烁二次后，进入STOP2模式
        4、按下任意触摸按键，唤醒MCU，D6闪烁三次后，再次进入SLEEP模式
        5、全程可通过串口工具查看相关信息
        
4、注意事项
    J41、J42、J43必须断开。
