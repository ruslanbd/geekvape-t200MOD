1、功能说明

    此例程展示了硬件CRC的基本功能和算法。
    
    其中有16bit校验与32bit校验。
    

2、使用环境

    软件开发环境：
        IDE工具：KEIL MDK-ARM 5.26.2.0
    
    硬件环境：
        开发板 N32G457QEL-EVB v1.2


3、使用说明
    
    1、时钟源：HSE+PLL
    2、主时钟：144MHz

4、注意事项
    无
